# gtuner

A Mobile Guitar Tuner.

## Dependencies
 - flutter_audio_capture
 - pitch_detector_dart
 - shared_preferences
 - permission_handler
 - google_mobile_ads
 - show_up_animation
 - numberpicker
 - google_fonts
 - just_audio

## Getting Started

### Get Packages

```bash
flutter pub get
```

### Run

```bash
flutter run
```
